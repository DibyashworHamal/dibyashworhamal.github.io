# Personal Portfolio Website 🚀

Welcome to the source code repository for my personal portfolio website! 

This website was designed and built from scratch to serve as my digital footprint, showcasing my professional background, skills, resume, and projects as an NEC Licensed Engineer.

> **🌐 Live Website:**[https://dibyashworhamal.com.np/](https://dibyashworhamal.com.np/)

## 🛠️ Tech Stack
This project was built without the use of heavy frameworks or drag-and-drop builders. 
* **Markup:** HTML5
* **Styling:** CSS3 (Custom responsive design)
* **Scripting:** Vanilla JavaScript
* **Editor:** VS Code

## ☁️ Deployment Architecture ($0 Cost Setup)
One of the main goals of this project was to establish a professional, secure, and fast web presence at zero recurring cost. Here is the architecture I used to achieve this:

* **Hosting:** [GitHub Pages](https://pages.github.com/) (Free static site hosting directly from this repository).
* **Domain Name:** Official `.com.np` domain registered via Mercantile Communications (Free for Nepalese citizens).
* **DNS Management & Security:** [Cloudflare](https://www.cloudflare.com/) (Used for DNS routing, proxying, and providing a free strict SSL/TLS certificate for HTTPS).

## ✨ Features
* **Fully Responsive:** Optimized for both desktop and mobile viewing.
* **Home:** Introduction and professional title.
* **About:** Detailed background and professional summary.
* **Resume:** Digital view of my education, experience, and qualifications.
* **Portfolio:** Showcase of my past projects and engineering work.
* **Contact:** Reach out to me directly.
